PEPSetWorkVecs

Sets a number of work vectors into a PEP object.

Synopsis

#include "slepcpep.h" 
PetscErrorCode PEPSetWorkVecs(PEP pep,PetscInt nw)
Collective on pep

Input Parameters

pep  - polynomial eigensolver context
nw  - number of work vectors to allocate
