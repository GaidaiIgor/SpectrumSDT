PEPCreate

Creates the default PEP context.

Synopsis

#include "slepcpep.h" 
PetscErrorCode PEPCreate(MPI_Comm comm,PEP *outpep)
Collective

Input Parameter

comm  - MPI communicator

Output Parameter

pep  - location to put the PEP context

Note

The default PEP type is PEPTOAR
