DSCond

Compute the inf-norm condition number of the first matrix as cond(A) = norm(A)*norm(inv(A)).

Synopsis

#include "slepcds.h" 
PetscErrorCode DSCond(DS ds,PetscReal *cond)
Not Collective

Input Parameters

ds  - the direct solver context
cond  - the computed condition number
