DSType

String with the name of the type of direct solver. Roughly, there are as many types as problem types are available within SLEPc.

Synopsis

typedef const char* DSType;
#define DSHEP    "hep"
#define DSNHEP   "nhep"
#define DSGHEP   "ghep"
#define DSGHIEP  "ghiep"
#define DSGNHEP  "gnhep"
#define DSSVD    "svd"
#define DSPEP    "pep"
#define DSNEP    "nep"
