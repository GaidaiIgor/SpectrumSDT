SVDReset

Resets the SVD context to the initial state (prior to setup) and destroys any allocated Vecs and Mats.

Synopsis

#include "slepcsvd.h" 
PetscErrorCode SVDReset(SVD svd)
Collective on svd

Input Parameter

svd  - singular value solver context obtained from SVDCreate()
