VecDuplicateEmpty

Creates a new vector of the same type as an existing vector, but without internal array.

Synopsis

#include "slepcvec.h" 
PetscErrorCode VecDuplicateEmpty(Vec v,Vec *newv)
Collective on v

Input parameters

v  - a vector to mimic

Output parameter

newv  - location to put new vector

Note

This is similar to VecDuplicate(), but the new vector does not have an internal array, so the intended usage is with VecPlaceArray().
