MFNReset

Resets the MFN context to the initial state (prior to setup) and destroys any allocated Vecs and Mats.

Synopsis

#include "slepcmfn.h" 
PetscErrorCode MFNReset(MFN mfn)
Collective on mfn

Input Parameter

mfn  - matrix function context obtained from MFNCreate()
