NEPGetTarget

Gets the value of the target.

Synopsis

#include "slepcnep.h" 
PetscErrorCode NEPGetTarget(NEP nep,PetscScalar* target)
Not Collective

Input Parameter

nep  - eigensolver context

Output Parameter

target  - the value of the target

Note

If the target was not set by the user, then zero is returned.
