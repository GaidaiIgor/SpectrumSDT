BVNormVecBegin

Starts a split phase norm computation.

Synopsis

#include "slepcbv.h" 
PetscErrorCode BVNormVecBegin(BV bv,Vec v,NormType type,PetscReal *val)

Input Parameters

bv  - basis vectors
v  - the vector
type  - the norm type
val  - the norm

Note

Each call to BVNormVecBegin() should be paired with a call to BVNormVecEnd().

See Also
