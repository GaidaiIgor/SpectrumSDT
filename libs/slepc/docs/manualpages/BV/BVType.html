BVType

String with the name of the type of BV. Each type differs in the way data is stored internally.

Synopsis

#define BVMAT        "mat"
#define BVSVEC       "svec"
#define BVVECS       "vecs"
#define BVCONTIGUOUS "contiguous"
#define BVTENSOR     "tensor"
