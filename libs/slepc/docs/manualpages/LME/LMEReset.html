LMEReset

Resets the LME context to the initial state (prior to setup) and destroys any allocated Vecs and Mats.

Synopsis

#include "slepclme.h" 
PetscErrorCode LMEReset(LME lme)
Collective on lme

Input Parameter

lme  - linear matrix equation context obtained from LMECreate()
