LMESetDimensions

Sets the dimension of the subspace to be used by the solver.

Synopsis

#include "slepclme.h" 
PetscErrorCode LMESetDimensions(LME lme,PetscInt ncv)
Logically Collective on lme

Input Parameters

lme  - the linear matrix equation solver context
ncv  - the maximum dimension of the subspace to be used by the solver

Options Database Keys

-lme_ncv <ncv>  - Sets the dimension of the subspace

Notes

Use PETSC_DEFAULT for ncv to assign a reasonably good value, which is dependent on the solution method.
