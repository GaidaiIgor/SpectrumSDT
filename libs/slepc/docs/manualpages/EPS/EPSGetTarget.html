EPSGetTarget

Gets the value of the target.

Synopsis

#include "slepceps.h" 
PetscErrorCode EPSGetTarget(EPS eps,PetscScalar* target)
Not Collective

Input Parameter

eps  - eigensolver context

Output Parameter

target  - the value of the target

Note

If the target was not set by the user, then zero is returned.
